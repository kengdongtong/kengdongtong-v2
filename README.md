# 坑洞通 2.0

## 全民回報，共同守護通行安全

坑洞通是一個由民眾自主發起的公益平台。

透過拍照、定位與社群協作，讓道路、人行道及公園步道等地面安全問題能被快速發現、公開記錄與持續追蹤，共同降低跌倒、車禍及用路風險。

---

## 回報範圍

### 🚗 道路類

* 道路坑洞
* 路面破損
* 路面龜裂
* 路面下陷
* 柏油剝落

### 🚶 人行道類

* 人行道坑洞
* 地磚破裂
* 地面隆起
* 步行危險區域

### 🌳 公園步道類

* 步道破損
* 地面塌陷
* 鋪面損壞
* 行走安全隱患

---

## 平台特色

* GPS 定位回報
* 現場照片上傳
* 地圖案件瀏覽
* 案件追蹤管理
* 行政區道路健康指數
* 熱門案件排行
* 社群確認機制
* 通知更新提醒

---

## 平台定位

本平台為民間公益資訊平台。

所有案件資訊由使用者自主提供，平台僅提供資訊整理、公開展示與追蹤服務。

本平台不隸屬任何政府機關、政黨或企業。

---

## 我們的願景

建立台灣最完整的道路與步行環境安全資料庫。

讓每一個坑洞都能被看見，
讓每一次回報都能促進改善，
讓每一位用路人都能更安全地通行。
